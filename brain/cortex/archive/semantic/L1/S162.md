Vericity transactions pipeline: S3-only data (4 CSVs 2022-2025, ~38MB), no dbo.transactions DB table exists. Schema includes transaction_date, price, area_apt, rooms, room_address, cadastre numbers, building details. Lambda produces Transactions.xlsx. Backend built at ~/vericity-transactions/ targeting Azure SQL inudat. Geocoding implemented via Nominatim with building-level dedup.
