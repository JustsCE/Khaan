When guiding a user on Windows (MINGW64/Git Bash/PowerShell): (1) confirm shell type before issuing commands, (2) never assume symlinks work — check Developer Mode or default to cp, (3) never assume python3 exists — check or use py/python, (4) issue one command per line to avoid bracket-paste concatenation.
