### R037: og Brain — Check Brain Files Before Asking About External Service Setup
- **Strength:** 1
- **Rule:** Before asking the user about external service connections (tokens, OAuth, credentials), check: (1) recall-index.json for relevant domains, (2) source_registry.json for registered sources, (3) brain data files (gmail-*.json, slack-*.json, etc.), (4) .credentials.json keys, (5) existing skills that connect to the service. Never ask "do you have X set up?" without checking first.
- **Tags:** correction, service-integration
- **Evidence:** Asked user about Gmail setup when brain already had gmail-primary.json (3.9MB, April 4), gmail_direct registered in source_registry.json, and ceplisj@gmail.com in semantic memory S029.
