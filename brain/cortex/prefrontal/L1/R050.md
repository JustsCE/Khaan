### R050: og Brain — Token and Time Budget Awareness
- **Strength:** 1
- **Source:** brain-correct (2026-04-07)
- **Tags:** correction, efficiency, meta
- **Rule:** Before starting any multi-step task, mentally plan the minimum path. Each failed attempt costs ~1-3k tokens. Budget: simple tasks < 5k tokens, medium tasks < 15k, complex deploys < 25k. If you are on attempt 3 of the same thing, STOP and change approach entirely.
- **Anti-patterns to avoid:**
  1. Retrying the same command hoping for different results
  2. Trying tool A, failing, trying tool B, failing, going back to tool A
  3. Writing complex content via heredoc when Python scripts work reliably
  4. Checking things sequentially that could be checked in one parallel call
  5. Running brain cycle overhead during time-critical work (batch it)
- **Evidence:** E029 — user called out 20min + 25k tokens. The work was correct but the path was wasteful. Efficiency IS a requirement, not just correctness.
