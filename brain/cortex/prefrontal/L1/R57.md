### R57: og Brain — MCP tools are session-bound
- **Added:** 2026-04-08 | **Strength:** 1 | **Source:** brain-retro
- **Tags:** correction, architecture
- **Rule:** When building automated pipelines (cron, CI, Lambda), NEVER rely on MCP. MCP requires active Claude session with OAuth. Use direct API calls with tokens.
- **Evidence:** Session 6: built MCP Slack pull, had to rewrite as standalone Python when user asked for CI/Lambda.
