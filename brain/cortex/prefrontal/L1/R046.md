### R046: og Brain — Minimize Separate Bash Calls — Combine Into Single Scripts
- **Strength:** 1
- **Rule:** When multiple Bash operations are needed, combine them into a single python -c script. Each separate Bash call risks a permission prompt. One combined call = one approval at most.
- **Exception:** Independent operations that truly need separate error handling.
- **Evidence:** Session 3 (2026-04-07): user frustrated by repeated permission prompts from mkdir, rm -rf, touch as separate calls. "PREMISSIONS!" (ALL CAPS).
