Rule: Before claiming a gating system (decision-engine, brain-cycle, or any hypothesis/cycle bin) is broken or deadlocked, READ the actual dispatcher exemption code in hook.py to confirm which tools are exempted and how the exemption pattern works. Both decision-engine and brain-cycle use the same pattern: the bin blocks all tools EXCEPT the specific skill or Bash subprocess that clears it. The mechanism itself is sound — failures originate from the skill instructions guiding the LLM to use individual tool calls instead of the required single subprocess invocation. Exception: If the dispatcher code itself has been recently modified (uncommitted changes or last commit < 1 hour), treat the mechanism as potentially broken and inspect the diff. Evidence: Previous brain-cycle deadlock was misdiagnosed as a dispatcher bug when the actual cause was brain-cycle.md skill instructions leading the LLM to invoke individual tools instead of a single Python subprocess, which the dispatcher correctly blocked.
