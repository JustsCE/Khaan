## Rule
When a subagent reports specific data findings (cell values, file contents, data states), verify the critical ones with a direct tool call before building a plan on them. Subagents can misread or misinterpret data.

## Evidence
Explore agent reported Melanu USA forecast cells as WHITE (non-blue) when they were actually BLUE (DCE6F1). Built an incorrect root cause analysis on this wrong finding. A single direct openpyxl check proved the agent wrong.
