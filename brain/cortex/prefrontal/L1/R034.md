### R034: og Brain — Never Use Inject Action — Only Block Works
- **Strength:** 1
- **Rule:** When adding rules to rules-engine.json, ONLY use action=block. Never inject. Inject rules (R028, THOUGHTS, RECALL_IDX) were all proven useless — Claude ignores injected context. If something is worth enforcing, block for it. If not worth blocking, it is not worth a rule.
- **Evidence:** R028 (Slack freshness inject — ignored), THOUGHTS (thoughts.jsonl inject — ignored), RECALL_IDX (recall-index inject — ignored). All three removed after proving ineffective.
