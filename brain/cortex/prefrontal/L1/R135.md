# R135: Agent prompts must use MCP tools, never curl

**Rule:** When writing agent trigger prompts, ALL external API calls must use MCP tools. Never use curl, fetch, or direct HTTP. The Anthropic sandbox blocks outbound HTTP.

**Evidence:** Solo Time agent was unable to reach dashboard API via curl for 3+ runs. Replaced with MCP tools and agents immediately worked.
