# R076: Resumable Functions Catch Up Naturally

When all pipeline functions are resumable (skip already-processed items), don't overthink scheduling or ordering. Just run them all on a regular interval and they will catch up over multiple cycles.

**Evidence:** Brain spent time analyzing whether 30-min gaps were enough for multi-run functions, proposing complex scheduling. User said: "the functions WILL catch up. Just start running them every 4h and after like 6 runs everything will be up-to-date."

**Rule:** If functions are resumable, set a simple recurring schedule and let them sort themselves out.
