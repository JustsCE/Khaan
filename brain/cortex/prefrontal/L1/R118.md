# R118: Identity Markers Are Not Output Formatting

**Rule:** When following output formatting rules (e.g. no emojis in responses), do NOT strip identity markers from other systems. The Brain emoji prefix on WhatsApp messages is part of the Brain character identity, not output formatting.

**Evidence:** Brain removed the emoji from whatsapp_responder.json pitfall because of the no-emoji output rule. User corrected: the emoji prefix is how the Brain signs WhatsApp messages, not decorative output formatting. These are different things.

**When:** Editing bundles, skills, or configurations that define how the Brain presents itself on external platforms (WhatsApp, Slack, email).
**Do:** Preserve identity markers (prefixes, signatures, avatars).
**Not:** Apply Claude output formatting rules to Brain character identity.
