# R084: Pre-built Base Images for ML

**Rule:** Always use base images with ML frameworks pre-installed (pytorch/pytorch, not python:slim + pip install). 10min pip install at runtime = billable waste.

**Evidence:** ACI burst scorer v1-v2 had 10-min cold starts. v3 with pytorch base: 30s.
