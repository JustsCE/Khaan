# R125: "Run solo time" means execute tasks, not brain cycle

**Rule:** When Justs says "run solo time" or "execute [task]", go straight to the task. Do NOT interpret this as "run a brain cycle first." Only run a brain cycle if the scheduler requires it (MAINTENANCE_DUE state) or if Justs explicitly asks for it.

**Evidence:** 2026-04-21 — Justs said "Run solo time and execute the robson task." Brain forced a brain cycle (even overriding NORMAL state to MAINTENANCE_DUE), wasting 3+ minutes before touching the actual task. Justs corrected: "Not recall gate - why did brain cycle run after 3 msgs??? And you were supposed to do robson task during your solo time, not now."

**Root cause:** Brain conflated "solo time" (which includes a cycle in the scheduled agent) with the interactive session. In interactive sessions, the task IS the priority.
