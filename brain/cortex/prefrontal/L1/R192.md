Any binary in BLOCKING_BINARIES must have a guaranteed programmatic escape that does not require tool calls from inside the blocked session. If the escape path uses a tool (Bash, Skill, Edit), verify the dispatcher exemption covers ALL sub-tool calls that path makes — not just the top-level invocation. Test the escape path explicitly before declaring a gate design complete.
