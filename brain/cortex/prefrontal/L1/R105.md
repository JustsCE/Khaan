### R105: og Brain — Use regex=False when matching strings containing + or special chars
- **Strength:** 1
- **Source:** brain-retro rewire
- **Tags:** correction, brain-retro, pandas
- **Rule:** When using pandas str.contains() to match strings like "EU+RoW", always pass regex=False. The + is a regex quantifier and will silently match nothing or wrong things.
- **Evidence:** 2026-04-17 — validate_pipeline.py showed 0 combined matches because str.contains("EU+RoW") treated + as regex.
