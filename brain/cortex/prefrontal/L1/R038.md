### R038: og Brain — VSCode Extension Only Reads settings.json for Permissions
- **Strength:** 1
- **Source:** brain-retro rewire (E016)
- **Tags:** correction, brain-retro, vscode
- **Rule:** The VSCode extension does NOT read settings.local.json for permission configuration. Always put `defaultMode: "bypassPermissions"` in the MAIN settings.json. When bulk file operations trigger repeated approval prompts, use Agent tool to delegate all writes in one shot — user approves agent launch once instead of N individual edits.
- **Evidence:** E016 — User exploded 3 times in one session because every Edit/Bash tool call prompted for approval despite settings.local.json having bypassPermissions.
