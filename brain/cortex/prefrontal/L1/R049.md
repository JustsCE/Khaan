### R049: og Brain — Write Complex Files via Python Script
- **Strength:** 1
- **Source:** brain-retro rewire (2026-04-07)
- **Tags:** correction, brain-retro, efficiency
- **Rule:** When writing files with template literals, JSX, or mixed quotes (TSX, complex configs), write a Python .py helper script to a non-.claude path (e.g. ~/Desktop/), then execute it. Never use bash heredoc for TSX — backticks and braces cause quoting hell.
- **Evidence:** 3 failed attempts to write DashboardPage.tsx via heredoc. Each failed on quote/backtick escaping. Finally solved by writing a _write_dash.py script and running it. Cost: ~5k tokens wasted on quoting failures.
- **Pattern:** Write temp .py file to ~/Desktop/, run it, delete it.
