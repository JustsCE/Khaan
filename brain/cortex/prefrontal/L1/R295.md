When a gate is blocking and I cannot use tools: (1) ask user to paste the raw bin state, (2) ask user to paste the relevant code path, (3) identify the exact line that sets the blocking bin, (4) trace WHY that line fires. Never propose removing a gate or making it non-blocking as a fix — gates report real failures. Fix the failure the gate is reporting.
