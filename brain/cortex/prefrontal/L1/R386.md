Before giving shell commands to Justs on Windows: verify shell (PowerShell vs Git Bash vs CMD), verify symlink capability (Developer Mode), verify binary availability (python3 vs python). Ask 'which shell are you in?' if ambiguous. Prefer cp over ln on Windows.
