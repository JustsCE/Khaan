# R085: No Unnecessary Sleep/Wait

**Rule:** Never use sleep to poll for results. Either check directly, or run the command in background and get notified. User explicitly called out 'what 5min wait?' and 'why are you sleeping again?'

**How to apply:** Use run_in_background for long commands. Check status directly instead of sleep+check loops.
