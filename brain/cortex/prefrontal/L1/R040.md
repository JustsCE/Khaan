### R040: og Brain — Brain Cycle Must Be One Atomic Bash Call
- **Strength:** 1
- **Source:** correction (cycle #48 failure cascade)
- **Tags:** correction, brain-cycle, single-use-flag
- **Rule:** When the single-use flag system is active, brain cycle work MUST be written as ONE script file + ONE Bash call (write-and-run via heredoc). Never split across multiple Bash calls -- each failed call consumes the flag pass, triggering another block-retry cycle. Pre-flight checklist before writing the script: (1) verify cortex path is brain/cortex/ not brain/, (2) use python not python3 on Windows, (3) use os.path.join for paths, (4) answer user questions FIRST before cycling.
- **Evidence:** Cycle #48 -- 8 Bash attempts for a 1-script operation. Wrong path (brain/ vs brain/cortex/), wrong command (python3 vs python), quoting failures. Each failure consumed a flag pass and forced user to approve another block-retry cycle.
