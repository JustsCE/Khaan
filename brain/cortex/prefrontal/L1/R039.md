### R039: og Brain — Use Agents for Bulk File Creation to Minimize Prompts
- **Strength:** 1
- **Source:** brain-retro rewire (E016)
- **Tags:** correction, brain-retro, workflow
- **Rule:** When creating 3+ files in one step, delegate to a single Agent instead of making individual Write/Edit calls. The user approves the agent launch once, and the agent creates all files without further prompts. Discovered when creating 9 MCP-related files — individual tool calls each triggered approval, but a single Agent call created all 9 with one approval.
- **Evidence:** E016 — 4 Dockerfiles + docker-compose + nginx + settings + tool_profiles + bundle = 9 files. Individual calls caused 3 user explosions before switching to Agent approach.
