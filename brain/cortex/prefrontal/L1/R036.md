### R036: og Brain — Always Use Imported Types in TypeScript Interfaces
- **Strength:** 1
- **Rule:** In TypeScript interfaces, never use typeof runtime.expression for types. Always import and use named type definitions. typeof only works at expression scope, not at interface definition scope.
- **Evidence:** NeuronDetail feature broke production build. Used typeof brain.nodes[0] in Props interface instead of NeuronNode import.
