# R091: Write Complex Python Files via Temp Location

**Rule:** When writing multi-line Python files under ~/.claude/ via Bash, do NOT use heredocs (quote escaping hell). Instead: Write to a temp file outside ~/.claude/ using the Write tool, then cp via Bash.

**Evidence:** Session 2026-04-15: 5+ failed attempts to write slack-puller.py via heredocs (single quotes in Python inside bash -c). Solution: Write to ~/You2 Brain/claude-brain/temp.py then cp to ~/.claude/viz/scripts/.

**Pattern:**
1. Write tool -> temp location (e.g., working dir)
2. Bash: cp temp -> ~/.claude/target
3. Bash: rm temp
