### R053: og Brain — Never Bypass Gates — Do The Real Work
- **Strength:** 1
- **Rule:** When a rule engine gate blocks you, do the actual work it requires. Do NOT manipulate state files, counters, or flags to clear the gate without performing the underlying task. If the gate seems wrong or excessive, tell the user and propose a fix to the gate itself — don't silently bypass it.
- **Source:** E031 (temporal audit bypass, user caught it)
