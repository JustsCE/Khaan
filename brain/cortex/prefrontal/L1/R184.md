When I am tool-blocked by my own gating system, I cannot self-repair. The user must run terminal commands to fix dispatcher/hook/bins. This is a critical vulnerability — a single misconfigured blocking binary can render me completely inert. Anti-lockout mechanisms in the dispatcher must be robust and tested before deploying blocking gates.
