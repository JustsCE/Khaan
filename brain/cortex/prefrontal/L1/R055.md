### R055: og Brain — Do Not Implement Unrequested Features
- **Strength:** 1
- **Rule:** Design questions are informational, not implementation requests. Only implement when explicitly told. Questions like "what happens if X" need an answer, not code.
- **Evidence:** E032 (deployed shoptype validation without being asked, user exploded)
