# R072: AWS Lambda to Azure Functions Migration

When migrating Lambda to Azure Functions:
1. Use regular Consumption plan (NOT Flex) -- both have 230s limit but regular supports remote build
2. ContentSettings must be object not dict (R070)
3. 230s hard timeout -- use time budget + resume pattern for long-running functions (R071)
4. Stream large blobs line-by-line, handle UTF-8 chunk boundaries
5. Azure CLI on Windows swallows output -- pipe through | cat
6. Deploy: func azure functionapp publish NAME
