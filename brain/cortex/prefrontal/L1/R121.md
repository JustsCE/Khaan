# R121: UTF-8 Encoding on Windows Python

**Rule:** Always pass encoding="utf-8" to open() when using Python on Windows. Default cp1252 silently truncates files with unicode.

**Why:** Brain edited backend/index.ts via python -c. File had unicode emoji. Python used cp1252 codec, wrote 0 bytes, reported success. File destroyed. Restored from git.

**How to apply:**
1. Every open(): encoding="utf-8"
2. Or prefer sed/heredoc for simple insertions
3. Only applies on Windows. Linux defaults to UTF-8.
