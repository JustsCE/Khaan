# og Brain — Beatrise Thesis — Results Table Column Conventions

## Chi² results table (X.N.2) nomenclature
- Effect-size column header: **"Krāmera-V"** (Latvian transliteration with hyphen). NOT "Cramér's V" or "Cramers V".

## Mann-Whitney results table (Y.N.2) — EXACTLY 6 columns
| Mainīgais | Virssvars Med (IQR) | Adipozi Med (IQR) | U | p | r_rb |

- Median formatted as **"Med (Q1–Q3)"**.
- **NO CI column** — explicit decision per Justs: "Interquartile range is a better metric".
- Effect size column is **r_rb** (rank-biserial), not Krāmera-V.

## Mann-Whitney validity table (Y.N.1) — EXACTLY 4 columns
| Mainīgais | n (Virssvars) | n (Adipozi) | Derīgs |

- NO "Min šūna" column (unlike chi² .1 which checks cell counts; MW validity is just group sizes).
