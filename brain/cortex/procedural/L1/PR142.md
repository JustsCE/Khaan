## Pattern
Long-press on a Three.js mesh inside react-three-fiber without @use-gesture.

## Mesh event handlers
- onPointerDown: record timestamp, start 400ms timeout
- onPointerUp / onPointerCancel: clear timeout
- If timer fires: set fired ref, invoke onLongPress
- onClick: if fired ref is true, swallow (disambiguate short vs long)
Also cancel on movement greater than 8px threshold.
