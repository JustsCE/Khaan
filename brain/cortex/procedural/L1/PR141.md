## When
Upgrading any React+Vite+Three.js page to a mobile-first feel.

## Stack
- framer-motion for UI animations + gestures + layout transitions
- @use-gesture/react for R3F canvas gestures beyond OrbitControls
- PWA manifest + 192/512 icons + viewport-fit=cover + theme-color + apple-mobile-web-app-*
- 100dvh with --app-vh fallback (lib/dvh.ts)

## Three.js tuning
- dpr [1, 1.5] on mobile; antialias off; powerPreference high-performance
- frameloop 'demand' when idle, 'always' when active
- Sphere segments 6->4 on mobile

## Primitives
- BottomSheet: 4 detents (hidden/peek/half/full), velocity-biased snap
- MobileTopBar: 48px + safe-area-top, whileTap buttons

## Haptics
tap=10, select=[8,40,8], success=[10,40,20], warn=5x pattern, heavy=30

<330KB gz target after framer-motion. Landed at 255KB on BrainPage.
