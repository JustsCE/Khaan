# og Brain — Dashboard Architecture -- Brain Pane + Notion Tables + Enhanced Digest

## Panes (goodhealth.lv/dashboard)
1. SLACK (blue) -- source panel with Notion-like table
2. WHATSAPP (green) -- source panel with Notion-like table
3. BRAIN (purple) -- cross-source view of PLAN/EXECUTE/PROJECT tasks
4. ALL TASKS (orange) -- read-only summary grouped by priority
5. DIGEST FEED (cyan) -- agent run summaries grouped by agent

## Brain Pane
- VIEW across all sources, not separate store
- GET /api/dashboard/brain-tasks scans all task files for PLAN/EXECUTE/PROJECT tags
- Groups: EXECUTE -> PLAN -> PROJECT
- PROJECT tasks show PhaseBadge (P1/P2/P3/TR/REV)

## Notion-like TaskRow
Columns: # | Task | From | Source | Tag | Actions
Compact 36px rows, hover highlight, sticky header.

## Enhanced Digest
Grouped by triggerName. Supports cortexEntries and identityUpdates fields.
