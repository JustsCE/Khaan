### PR086: og Brain — Anthropic Routine — Commit to Main (Not Feature Branches)
- **Domain:** Infrastructure / Routines
- **Learned:** 2026-04-17
- **Strength:** 1
- **Source:** brain-retro

**Cue:** Creating or updating an Anthropic Routine (RemoteTrigger) that commits to a repo.

**Routine:**
1. In job_config.session_context.outcomes, set branches to ["main"] not ["claude/feature-name"]
2. In the prompt, explicitly state: "Commit and push directly to main. Do NOT create feature branches."
3. Add ALL repos the agent needs to BOTH sources AND outcomes arrays
- **Gotcha:** Default behavior creates a unique branch per run (e.g. claude/teacher-run-G35en) — work gets orphaned
- **Gotcha:** Each run gets a fresh environment — branches from previous runs are not visible
- **Gotcha:** If multiple repos needed, add each as a separate object in both sources[] and outcomes[]
- **Verification:** After run, check git log main for the commit (not git branch -r)
- **Apply when:** Setting up any Anthropic Routine that writes to git repos
