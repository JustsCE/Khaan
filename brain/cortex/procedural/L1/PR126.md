# og Brain — Chapter-opening X.0 subsection convention (Latvian thesis)

In Latvian medical thesis chapters, the X.0 subsection is the chapter-opening intro/setup, and X.1+ is the actual content.

Examples from Beatrise thesis:
- 3.0 "Pētījuma mērķis un iekļautie faktori" (study purpose + Robson classification table)
- 4.0 "Aprakstošie dati" (descriptive overview before test results in 4.1+)

Pattern: X.0 = setup, X.1 onwards = analysis.
