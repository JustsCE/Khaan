# PR099: EC2 Swap File Setup

**Cue:** EC2 t3.small OOM kills, or any small instance without swap.

**Routine:**
1. SSH into instance
2. Create swap file:
   ```bash
   sudo fallocate -l 2G /swapfile
   sudo chmod 600 /swapfile
   sudo mkswap /swapfile
   sudo swapon /swapfile
   ```
3. Make persistent across reboots:
   ```bash
   echo "/swapfile none swap sw 0 0" | sudo tee -a /etc/fstab
   ```
4. Set swappiness (prefer RAM, swap only under pressure):
   ```bash
   sudo sysctl vm.swappiness=10
   echo "vm.swappiness=10" | sudo tee -a /etc/sysctl.conf
   ```

**Verification:** `swapon --show` shows /swapfile, `free -m` shows swap total.
**Prerequisites:** Enough disk space (df -h, need 2x swap size free).
**Gotcha:** Swap prevents OOM kills but does NOT prevent MCP container exhaustion from sustained long requests (see E061 incident #3).
**Reward:** System has 2GB RAM + 2GB swap = 4GB effective memory. OOM kills eliminated.
