# og Brain — Discussion-section handoff format — 100% bullets, NO prose, per Robson group

Convention for HANDOFF-to-collaborator versions of statistics-heavy thesis sections:
- Discussion section = 100% bullet points, NO prose paragraphs
- Organized BY ROBSON GROUP (not by thematic cluster — that was rejected)
- Order within section 5: per-group sections (R1, R2A, R2A1, R2A2, R3, R4, R5) → Starpgrupu atradumi → Pētījuma trūkumi
- NO literature comparison, NO hedging paragraphs, NO clinical implications in handoff
