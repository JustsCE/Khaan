## Bulk Disable / Pause (added 2026-04-26)

To pause many or all triggers (e.g. when the user wants "kill all agents"):

1. `RemoteTrigger {action:"list"}` — returns all triggers with current `enabled` state.
2. For each trigger to pause: `RemoteTrigger {action:"update", trigger_id:"<id>", body:{enabled:false}}`.
3. **The API has no delete operation.** Direct user to https://claude.ai/code/scheduled for permanent removal.
4. In-flight runs continue to completion; only future schedule firings stop.
5. To resume: same update call with `{enabled:true}`.
