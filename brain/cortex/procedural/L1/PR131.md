# og Brain — MCP dashboard task tools in brain-mcp server

4 tools added to viz/mcp-servers/whatsapp/server.py:
- get_dashboard_tasks(source) -> GET /api/dashboard/tasks/:source
- create_dashboard_task(source, title, ...) -> POST /api/dashboard/tasks/:source
- update_dashboard_task(source, task_id, ...) -> PATCH /api/dashboard/tasks/:source/:id
- complete_dashboard_task(source, task_id, actor) -> POST /api/dashboard/tasks/:source/:id/done

All proxy to backend at http://brain-viz:3001 with Bearer changeme auth.
Solves sandbox curl-blocking: remote agents use MCP tools instead of curl.
