# og Brain — Project Execution Pipeline -- 4-Phase Autonomous Work System

## Overview
PROJECT-tagged dashboard tasks flow through a 4-phase pipeline:
1. Brain solo-time writes draft 1 (PLANNING_1)
2. Brain tests + writes draft 2 (PLANNING_2)
3. Brain finalizes + submits for Teacher review (PLANNING_3 -> TEACHER_REVIEW)
4. Teacher approves or requests revision

## Key Files
- brain/project-engine.py -- CLI state machine (init/advance/submit-review/verdict)
- brain/projects/{task_id}/state.json -- phase tracking per project
- brain/projects/{task_id}/plan.md -- living plan document
- viz/backend/src/index.ts -- 6 API endpoints (/api/dashboard/projects/*)
- viz/mcp-servers/whatsapp/server.py -- 6 MCP tools

## MCP Tools
get_active_projects, get_project_plan, init_project, advance_project, submit_project_review, project_verdict

## Solo-Time Integration
Step 0 in PHASE 2. PROJECTs take priority over EXECUTE/PLAN. Route by status.

## Teacher Integration
PRIORITY 1 in Teacher prompt. Evaluates: completeness, feasibility, risk, charter, quality.
