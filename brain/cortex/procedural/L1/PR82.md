## generate_report.py

HTML report generator pulling from S3 actuals + forecast data.

### Key Design Decisions
- Date-based actuals/forecast split (not week-based) to avoid CW dip/spike artifacts
- Shoptype filter for undefined values
- --exclude-products / --only-products CLI flags
- CW16+ label for forecast tab

### Output
8 charts + 10 deep dive cards per report.
