# PR073: Context Builder

Push-triggered GH Actions workflow that runs a Claude agent to distill raw sync data into accumulated context files.

## Trigger
Push to brain/sync-state/slack-conversations.json or brain/sync-state/fireflies-transcripts.json

## Output
- brain/sync-state/slack-context.json: active threads, people activity, pending actions, topic trends
- brain/sync-state/fireflies-context.json: recent decisions, open action items, meeting patterns, person insights

.github/workflows/context-builder.yml
