# PR107: Dashboard Task and Digest Cleanup

**Cue:** Stale tasks or digests from disabled agents cluttering the dashboard.

**Routine:**
1. List tasks per source:
   ```bash
   curl -s "https://goodhealth.lv/api/dashboard/tasks/SOURCE"
   ```
   Where SOURCE = whatsapp, slack, fireflies
2. Scrap each stale task:
   ```bash
   curl -s -X POST "https://goodhealth.lv/api/dashboard/tasks/SOURCE/TASK_ID/scrap"
   ```
3. Clear digests (stored in container filesystem):
   ```bash
   docker exec brain-viz-brain-viz-1 sh -c 'echo "[]" > /brain/dashboard/digests.json'
   ```

**Gotcha:** There is no DELETE endpoint for digests — must overwrite the file inside the container.
**Gotcha:** Task sources are lowercase: whatsapp, slack, fireflies.
**Gotcha:** The `/api/dashboard` URL returns frontend HTML, not JSON. Use `/api/dashboard/tasks/:source` for the API.

**Verification:** `curl -s https://goodhealth.lv/api/dashboard/digests` returns `[]`.
