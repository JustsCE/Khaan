# PR100: /connect Page for Remote Control Access

**Cue:** Need to connect to the Brain from any device.

**Routine:**
1. Open https://goodhealth.lv/connect
2. Page shows current env link with status indicator
3. Click CONNECT TO BRAIN to open Claude Code on EC2
4. Auto-refreshes 30s. Copy button available.

**Architecture:**
- ExecStartPost polls journalctl, writes env.json to /home/ubuntu/brain-shared/
- Docker bind mount exposes to brain-viz at /app/shared:ro
- GET /api/remote-control serves it
- React ConnectPage at /connect (hidden from nav)
