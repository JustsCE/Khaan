# PR094: Restart Remote Control After MCP Changes

**Cue:** Added new MCP server to settings.json and user needs it in iOS/web sessions.

**Routine:**
1. Verify MCP entry in ~/.claude/settings.json
2. Restart service: sudo systemctl restart claude-brain.service
3. Wait 3 seconds, then get the new environment link:
   journalctl -u claude-brain.service --no-pager -n 10 | grep claude.ai
4. Give user the https://claude.ai/code?environment=env_XXXX link
5. User opens link on iOS/web to get a new session with updated MCPs

**Why restart:** The remote-control process reads settings.json at startup. New MCP entries added after startup are not picked up by existing sessions.

**Verification:** New session shows MCP tools in tool list.
