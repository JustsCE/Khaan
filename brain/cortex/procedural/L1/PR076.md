# PR076: EC2 Remote Control Setup

**Cue:** Setting up Claude Code Remote Control on goodhealth.lv EC2 for always-on iOS access.

**Routine:**
1. SSH into EC2: ssh -i ~/Desktop/good_health_machine.pem ubuntu@goodhealth.lv
2. Install Node.js 20: curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash - && sudo apt-get install -y nodejs
3. Install Claude Code: sudo npm install -g @anthropic-ai/claude-code
4. Clone brain repo to ~/.claude/ using PAT-authenticated URL from /brain/ remote
5. Generate Linux settings.json: replace Windows paths with /home/ubuntu/.claude/, python with python3
6. Create settings.local.json in ~/.claude/.claude/ with bypassPermissions
7. Accept workspace trust in ~/.claude.json: set projects["/home/ubuntu/.claude"]["hasTrustDialogAccepted"] = True
   - Gotcha: Trust is in ~/.claude.json NOT settings.local.json
   - Gotcha: Trust is PER-DIRECTORY
8. Create systemd service: ExecStart=/bin/bash -c 'echo y | /usr/bin/claude remote-control --name brain'
   - Gotcha: Must pipe echo y for non-interactive Enable Remote Control prompt
9. systemctl daemon-reload, enable, start
10. Connect from iOS Claude app or claude.ai/code

**Prerequisites:** EC2 with Docker (goodhealth.lv), PEM key, GitHub PAT
**Verification:** systemctl status shows active(running), iOS app connects
**Reward:** 24/7 Brain access from any device. ~347MB RAM on t3.small.
