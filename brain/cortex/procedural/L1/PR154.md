# PR154 -- og Brain — Triage Dedup: Same-Initiative Grouping

**Created:** 2026-04-27
**Status:** active
**Strength:** 1
**Reinforced:** 0
**Domain:** procedural (triage)

## Rule

When 3+ tasks from the same source meeting share **≥2** of:
- (a) same people
- (b) same project / initiative keyword
- (c) created in the same triage batch

…flag as duplicates of one initiative. Keep the broadest-scope task; merge specifics into its description.

## Pattern signals

- **Same source meeting** (Fireflies, Slack thread, etc.)
- **Same people** in all entries
- **Same domain object / initiative** (different verbs, same noun: "build / document / schedule X")
- **Temporal cluster** (created in same batch)
- **Verb diversity, noun unity** — different actions, same target

## Applies to

- Fireflies-derived task triage
- Slack-derived task triage
- Manual user grouping confirmations
