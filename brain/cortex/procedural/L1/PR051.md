## Structure
- /global (GlobalMap.tsx): 3D WebGL globe via three.js/@react-three/fiber. Rotating earth with active (green pulse) and target (indigo) market markers. Markets: Riga (active), Tallinn, Vilnius, Warsaw, Berlin, Helsinki (target). Market list panel on right.
- /:country (CountryMap.tsx): Leaflet map with country GeoJSON outline (fetched from GitHub datasets CDN). City pinpoints with SVG markers. Latvia config: Riga (active), Jurmala, Liepaja, Daugavpils (coming soon). Back nav to /global.
- /:country/:city (CityMap.tsx): Cadastre district map (merged polygons from /riga-districts.geojson). Default zoom = config.zoom + 1 (2x closer). Districts clickable -> /search. Back nav to country.
- React Router v6 ranked matching: static routes (/chat, /admin) always win over /:country params.
- Three.js chunk ~915KB but lazy-loaded.
