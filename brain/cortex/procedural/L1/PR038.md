### PR038: og Brain — Forecast Shoptype Review - Formula-Based Forecasts
- **Stage:** conscious
- **Learned:** 2026-04-09
- **Executions:** 2
- **Strength:** 1
- **Cue:** Converting forecast shoptype review Excel from static values to interactive formulas
- **Routine:**
  1. Shop forecast row: =IFERROR((1+F-YoY pct)*Actuals LY, 0) for forecast months only
     - Gotcha: Only write formula when fc_val != 0, otherwise write na to avoid LY fill-in
  2. F-YoY pct row: Static value from (forecast/actuals_ly - 1), user can edit
  3. Total Forecast row: =SUM(all shop forecast rows) for forecast months
  4. Total F-YoY pct: =IFERROR(Forecast/Actuals LY - 1, 0)
  5. Other shoptype row: fc_override_keys includes both Other shoptype and shoptype-level forecasts
  6. write_top5_shops_block returns (r, shop_forecast_rows) for SUM formula references
  7. Product block written first (static), then top5 block, then overwrite product rows with formulas
- **Prerequisites:** actuals.csv and seperate_forecasts_combined.csv in S3
- **Verification:** python tests/run_local.py review-shoptype --event with shoptype Enterprise
- **Reward:** Interactive Excel where users change F-YoY pct and forecasts recalculate
