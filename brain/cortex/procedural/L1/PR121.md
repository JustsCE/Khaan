# og Brain — LR Variable Exclusion Reasons — Use the Right Term

When writing methodology tables that list excluded predictors, the three distinct reasons must not be conflated:

## 1. Post-outcome / conditioning on a collider
Variable is only defined for one arm of the outcome (e.g. labour-stage durations exist only for vaginal deliveries; NaN for emergency CS). Including it causes **selection bias** — model fits a self-selected sub-sample only.

Correct wording: "Mainīgais ir definēts tikai tām pacientēm, kurām dzemdības noritēja dabiski — iekļaušana radītu atlases novirzi."

DO NOT call this "immortal-time bias" — that's a different, cohort-study concept about time-varying exposure.

## 2. Mediator on causal pathway
Variable is caused BY the exposure and in turn causes the outcome (e.g. maternal KMI -> fetal macrosomia -> CS risk). Adjusting for it **blocks part of the exposure's total effect**, under-estimating the answer to "what is the total effect of KMI on CS?"

Correct wording: "Mediators — KMI ietekme uz augla svaru, kas savukart ietekme risku. Ieklausana blokeetu dalu no KMI kopeejas ietekmes."

Not a temporal issue — birth weight is recorded for every delivery. It's a causal-graph issue.

## 3. Captured by stratification (constant within stratum)
Variable is a constant within every stratum of the primary analysis (e.g. Reta is 0 for Robson 1-4 by nullipara/non-scar definition; 1 for Robson 5). Mathematically collinear with intercept; can't be fit.

Correct wording: "Stratifikacija jau pilnibaa fikse so mainigo — tas ir konstants katra Robsona grupa."

## Rule
When writing the exclusion table, ask: is this variable (a) defined on only one outcome arm -> collider, (b) downstream of exposure -> mediator, or (c) constant within stratum -> captured? Each has a different fix and different wording.
