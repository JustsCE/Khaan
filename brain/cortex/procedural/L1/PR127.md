# og Brain — Study-purpose paragraph template (Latvian medical thesis §X.0 opener)

Canonical 5-part structure:
1. Goal: "Pētījuma mērķis ir raksturot/izskaidrot [factor] ietekmi uz [outcome] [scope]"
2. Design: "autors veic retrospektīvu analīzi, balstoties uz [data source]"
3. Period: "periodā no [start date] līdz [end date]"
4. Dataset: "datu kopa sastāv no N pacientēm, no kurām X ([Y] %) ..."
5. Data source + ethics: "anonimizēta medicīniskā dokumentācija un dzemdību vēstures, bez iespējas atsaukties uz konkrētu personu"
