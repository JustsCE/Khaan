### PR057: og Brain — Vericity ACR Build on Windows
- **Domain:** Vericity / Deployment
- **Learned:** 2026-04-11
- **Reinforced:** 0 times
- **Strength:** 1
- **Facts:**
  - ACR build on Windows requires --no-logs flag to avoid Unicode crash during log streaming
  - Tag scheme is sequential (v13, v14...v18)
  - Builds take ~4 min on ACR
  - Full deploy: az acr build --no-logs, then az containerapp update
  - Three concurrent builds submitted when first attempts appeared to hang (context upload was slow)
- **Apply when:** Deploying Vericity to Azure Container Apps from Windows
