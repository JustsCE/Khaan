# PR102 -- Autonomy Engine v2 Task Lifecycle

## Cue
Brain needs to detect, classify, and execute tasks from WhatsApp/Slack/Fireflies autonomously.

## Routine
1. **Task Input** (during source agent runs: WhatsApp, Slack, Fireflies):
   - Detect actionable messages (Hey Brain!, direct requests)
   - Classify: EXECUTE (simple, <5min) or PLAN (complex, needs clarification)
   - POST to /api/dashboard/tasks/:source with tags ["source", "EXECUTE"/"PLAN"]
   - Check PENDING tasks for answers, PATCH to EXECUTE if found

2. **Task Execution** (during Brain Solo Time agent):
   - Fetch all open tasks with autonomy tags from dashboard API
   - Charter alignment check before each task
   - EXECUTE tasks: do it, mark done (max 3/run)
   - PLAN tasks: create plan + 3 questions, PATCH to PENDING, send questions to source (max 2/run)
   - PENDING with answers: execute plan, mark done

3. **State machine**: EXECUTE->done | PLAN->PENDING->EXECUTE->done

## Key endpoints
- POST /api/dashboard/tasks/:source — create task
- PATCH /api/dashboard/tasks/:source/:id — update tags/description/title
- POST /api/dashboard/tasks/:source/:id/done — mark complete

## Kill switch
brain/autonomy-pause file. All agents check before acting.
