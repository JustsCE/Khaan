### PR032: og Brain — AWS Lambda Create & Deploy via boto3
- **Stage:** conscious
- **Learned:** 2026-04-08
- **Executions:** 3
- **Strength:** 1
- **Cue:** Need to create/deploy Lambda, AWS CLI unavailable or SSO expired
- **Routine:**
  1. Get config from existing function: lam.get_function()
  2. Zip: zipfile.ZipFile(buf) -> zf.write(lambda_function.py)
  3. Create: lam.create_function() or Update: lam.update_function_code()
  4. Wait 3-5s for propagation before testing
  5. Gotcha: IAM role scoped to original function log group, add inline policy
  6. Gotcha: Role needs lambda:InvokeFunction if calling other Lambdas
- **Prerequisites:** boto3, valid AWS creds (SSO expires, ask user)
- **Verification:** lam.invoke(RequestResponse) and check payload
