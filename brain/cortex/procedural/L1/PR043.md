# Follow User Instructions Directly

When user says "check X" or "look at X", go to X DIRECTLY. Do not:
- Explore tangentially
- Search repo trees when told to check secrets
- Try multiple alternative approaches before the one instructed

Follow the literal instruction first. Explore only if that fails.

**Origin:** Brain wasted 10+ tool calls exploring repo file trees when user explicitly said to check GitHub secrets. Caused extreme frustration.
