# og Brain — Beatrise Thesis docx — Formatting Rules

## Never use Word built-in Heading 1 / Heading 2 styles
Default renders as Calibri 16pt blue — breaks visual consistency with Justs' custom Times New Roman template.

**Wrong:**
```python
p = doc.add_paragraph(style='Heading 1')
r = p.add_run(text); r.font.name = 'Times New Roman'
```

**Right:**
```python
p = doc.add_paragraph(style='Normal')
r = p.add_run(text)
r.bold = True
r.font.size = Pt(13)
r.font.name = 'Times New Roman'
```

## Approved styles to use
- 'Normal' — body text, Times New Roman 11pt
- 'Compact' — dense paragraph style inside table cells, Times New Roman 10pt
- 'Table' — custom table style with bottom border under header only

## Approved bullet format
Not 'List Bullet' (Word default rendering is inconsistent). Instead use:
```python
p = doc.add_paragraph(style='Normal')
p.paragraph_format.left_indent = Cm(0.75)
p.paragraph_format.first_line_indent = Cm(-0.5)
r = p.add_run("•  " + text)
r.font.name = 'Times New Roman'; r.font.size = Pt(11)
```

## Discussion section structure
Bullets MUST be organized BY ROBSON GROUP (R1, R2A, R2A1, R2A2, R3, R4, R5 + cross-cutting + limitations), NOT by thematic cluster.

Justs corrected: thematic organization ("Blakusslimību izplatība", "Dzemdību iznākums", etc.) → group-based ("Robsona 2A grupa", "Robsona 4 grupa", etc.).

Rationale: the whole thesis framework is Robson-stratified; discussion must mirror that stratification.
