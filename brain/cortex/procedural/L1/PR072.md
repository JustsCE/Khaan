# PR072: Fireflies Puller

## API
- Endpoint: https://api.fireflies.ai/graphql
- Auth: Bearer token (FIREFLIES_API_KEY)
- Key query: transcripts(fromDate, limit, skip)
- Summary fields: overview, shorthand_bullet (NOT bullets), action_items

## Delta Sync
1. Read fireflies-pull-state.json for knownIds
2. Query transcripts with fromDate filter (3-week window)
3. Skip known IDs, fetch summaries for new ones
4. Merge, trim, write state

## Files
- viz/scripts/fireflies-puller.py (~10s runtime)
- .github/workflows/fireflies-puller.yml (2h cron at :47)
- brain/sync-state/fireflies-pull-state.json (state)
- brain/sync-state/fireflies-transcripts.json (data)
