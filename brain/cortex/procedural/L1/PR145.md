# og Brain — PR145: Isolated wa-bridge Container Deploy

## Cue
You changed viz/whatsapp-bridge/src/ and need to deploy on goodhealth.lv EC2.
The brain-control Lambda's deploy-viz action only handles brain-viz frontend +
backend — it does NOT rebuild the wa-bridge container. Manual deploy required.

## Routine

### 1. Edit + commit the source in the brain repo
- Edit ~/.claude/viz/whatsapp-bridge/src/index.ts (or other files)
- `cd ~/.claude && git add viz/whatsapp-bridge/src/ && git commit -m "wa-bridge: ..." && git push`

### 2. Sync to /opt/brain-viz (the docker-compose build context)
- `cp ~/.claude/viz/whatsapp-bridge/src/index.ts /opt/brain-viz/whatsapp-bridge/src/index.ts`
- Or `rsync -av ~/.claude/viz/whatsapp-bridge/src/ /opt/brain-viz/whatsapp-bridge/src/`
- The /brain volume mount is the OLD copy and is NOT used at build time. Build context is /opt/brain-viz/whatsapp-bridge/ on the host.

### 3. Rebuild + restart ONLY wa-bridge (R146-isolated)
- `cd /opt/brain-viz && docker compose build wa-bridge`
- `cd /opt/brain-viz && docker compose up -d wa-bridge`
- Service name is `wa-bridge` (not brain-viz-wa-bridge — thats the container name with project prefix).

### 4. Verify
- `sleep 8 && docker logs --tail 20 brain-viz-wa-bridge-1`
- expect "Connected to WhatsApp"

## Prerequisites
- Code change committed + pushed in the brain repo
- /opt/brain-viz/whatsapp-bridge/ exists on the EC2 with build context
- Docker daemon running, brain-viz_wa-auth volume present (preserves WhatsApp session)

## Gotchas
- WhatsApp auth is in wa-auth docker volume, preserved across rebuilds
- Bridge has 5000+ persisted messages and 70+ buffered — these survive rebuild
- The deploy-viz Lambda action only handles brain-viz, NOT wa-bridge
- NEVER run bare `docker compose build` or `docker compose up -d` (R146)

## Verification
- Container "Up X seconds" in docker ps
- "Connected to WhatsApp" in logs
- Send a test message with /brain-execute and confirm `[wa-task] EXECUTE: ...` log

## Reward
wa-bridge runs the new code, WhatsApp session unbroken.
