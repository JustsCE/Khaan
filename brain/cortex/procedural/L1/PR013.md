### PR013: og Brain — Goodhealth.lv Viz Deployment Pipeline
- **Stage:** conscious
- **Learned:** 2026-04-04
- **Executions:** 16
- **Strength:** 1
- **Cue:** Push to viz/** in claude-brain repo
- **Routine:**
  1. Push code to main branch (changes under viz/)
  2. deploy-viz.yml GitHub Action triggers automatically
  3. Action SSHs to EC2, clones repo, copies viz/* to /opt/brain-viz/
  4. docker compose build --no-cache + docker compose up -d (2-3 min downtime)
  5. nginx-active.conf is IMMUTABLE on server (chattr +i) — deploy cannot overwrite it
  6. Brain webhook also fires: brain-webhook.yml POSTs to /api/webhook/github for git pull
  - Gotcha: deploy causes 2-3 min downtime because it does docker compose down BEFORE build
  - Gotcha: cert check ran as ubuntu user who cant read /etc/letsencrypt — used to break HTTPS every deploy. Fixed with immutable config.
- **Prerequisites:** EC2_SSH_KEY + EC2_USER + BRAIN_PAT secrets in GitHub repo. PEM key at ~/Desktop/good_health_machine.pem for manual SSH.
- **Verification:** curl -s https://goodhealth.lv/api/health returns ok:true with updated cycleCount
- **Reward:** Viz live at goodhealth.lv with latest brain state
- **Errors:** HTTPS broke on every deploy until nginx-active.conf was made immutable (chattr +i)
  - Gotcha: NEVER commit viz/frontend/dist/ — Docker builds from source. Committed dist with stale hashes = blank page in production (E080).
