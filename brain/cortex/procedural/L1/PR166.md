# PR166: Spec a brain engine doc that obeys R164

**Cue:** writing or editing any JustsCE/Brain v2 engine doc.

**Routine:**
1. Open the engine HTML doc.
2. Before writing any prose, ask: am I describing THIS engine, or am I describing the system around it? If the latter, delete.
3. Pre-flight grep before commit. Forbidden tokens: count words, older-design narration, comparative phrasing, every other engine name except in the Inputs table.
4. Cross-engine references allowed only in: Inputs table, binary readers note, Out-of-scope bullet list (max 3-4 bare items).
5. After commit, re-grep across forbidden tokens. Any matches outside permitted zones, revise.

**Prerequisites:** engine scope is clear before drafting; other engine docs accessible for cross-checking.

**Verification:** forbidden-token grep returns 0 in body prose; every cross-engine name appears only in Inputs table cells or Out-of-scope bullets.

**Reward:** engine doc passes the R164 grep gate. Modular spec.

**Source incident:** Brain v2 design 2026-05-05. R164 violations recurred 5+ times across multiple engine docs. Fix is the same every time: name only the engine being specified.
