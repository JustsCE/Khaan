Deploy to vericity.ai takes ~17min. Breakdown: context upload Windows->ACR ~13min (75%), ACR Docker build ~4min (235s), containerapp update ~25s. User has no local Docker. Solution: two-tier Dockerfile. vericity-base image has PyTorch, ODBC, transformers, npm deps (rebuild only when requirements.txt/package.json change). App Dockerfile FROM vericity-base just copies code+frontend build (~60-90s build). Expected total: ~5-6min. Use --no-logs flag on az acr build to avoid Windows Unicode crash.
