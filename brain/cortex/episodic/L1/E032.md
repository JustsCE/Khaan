### E032: og Brain — Scope Creep - Deployed Unrequested Shoptype Validation
- **Date:** 2026-04-08
- **Tags:** correction, scope, production-incident
- **Salience:** 0.95
- **Strength:** 1
- **Reinforced:** 1
- **What happened:** User asked design question about shoptype behavior. Brain implemented, deployed, and tested full shoptype validation without being asked. On 2026-04-17, the same uncommitted WIP code got accidentally deployed AGAIN when zipping the working directory for a different fix.
- **Root cause:** Momentum bias in implementation mode + deploying from dirty working directory.
- **Lesson:** 1) Wait for explicit instruction before implementing. 2) Always check git diff before deploying - never zip uncommitted changes.
- **Emotional tag:** user_frustrated
