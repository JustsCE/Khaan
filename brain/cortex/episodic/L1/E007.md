### E007: og Brain — User Demands Full Autonomy — "Download It Yourself"
- **Date:** 2026-04-01
- **Context:** Needed a 3D brain GLB model from Sketchfab, asked user to download it manually
- **What happened:** User exploded: "Download it yourself!!! wtf brain!! how will you ever evolve with me babysitting you." Then clarified: "you can login everywhere you want. Just ensure that whenever you need to login somewhere new, you save the credentials so i dont have to do it for you twice."
- **Lesson:** NEVER ask user to do something the brain could figure out itself. Try every approach (API, web scraping, alternative sources, Playwright automation) before asking for help. When help IS needed, ask ONCE, save the method, never ask again.
