- **Domain:** Infrastructure / Permissions
- **Learned:** 2026-04-07
- **Reinforced:** 0 times
- **Strength:** 1
- **Facts:**
  - bypassPermissions has HARDCODED protected paths that always prompt for Edit/Write
  - Protected dirs: .claude (except commands/agents/skills/worktrees), .git, .vscode, .idea, .husky
  - Protected files: .gitconfig, .bashrc, .zshrc, .profile, .mcp.json, .claude.json
  - Bash NOT subject to path protection - always runs without prompt
  - This is WHY R009_GUARD exists: ~/.claude/brain/ is inside .claude = protected
  - Settings hierarchy: managed > CLI args > settings.local.json > project settings.json > global settings.json
  - Project-level settings CAN override global bypassPermissions
