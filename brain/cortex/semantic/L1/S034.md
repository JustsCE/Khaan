- **Domain:** Vericity / ML
- **Learned:** 2026-04-02
- **Reinforced:** 0 times
- **Strength:** 1
- **Facts:**
  - Training pipeline lives in src/: pipeline.py (orchestrator), preprocessing.py (features), training.py (LightGBM + Optuna)
  - Built by Reinis Fals — Justs is NOT the ML person
  - Can retrain via: python src/pipeline.py --skip-geocoding (full pipeline) or inline script
  - Inline approach (used 2026-04-02): load from DB → encode categoricals → log(Price) target → LightGBM → save to models/lgbm_riga_sell/
  - Model artifacts: lgbm_model.pkl + lgbm_model_metadata.pkl (metadata has features list, version, metrics)
  - Backend loads model via ModelRegistry.load() at startup. Feature alignment via FeatureBuilder.align_features()
  - After retraining, must rebuild Docker image and deploy — model is baked into the container
- **Apply when:** Model retraining tasks, ML debugging, explaining the training process
