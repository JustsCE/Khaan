### S037: og Brain — RemoteTrigger API + Active Monitoring Vision
- **Domain:** Infrastructure / Automation
- **Learned:** 2026-04-07
- **Reinforced:** 1 time
- **Strength:** 1
- **Facts:**
  - RemoteTrigger: Anthropic-hosted headless Claude agents with cron schedules
  - API: create/list/get/update/run via RemoteTrigger tool (OAuth auto-injected)
  - Runs 24/7 independent of CLI session, has access to MCP tools (Slack, Gmail)
  - Alternative: CronCreate (session-only, 7-day max, dies on session close)
  - User vision: multiple cadence triggers (30min Gmail, 2h Slack, 24h daily briefing)
  - Each trigger: pull latest data, create summary + action points, send to user
  - Delivery options: Slack DM, Claude iOS app, goodhealth.lv
  - This is the next major brain capability: autonomous monitoring without user initiating
  - Agent types: general-purpose (full tools), explore (read-only), plan (read-only), claude-code-guide (docs), verification
  - Each subagent = separate API conversation with own context window and restricted tools
  - Pricing: Opus 5/5 per M tokens, Sonnet same, Haiku /. Subagents consume tokens independently
  - Claude Code subscription (Max/Pro) includes agent usage. API key = pay per token
