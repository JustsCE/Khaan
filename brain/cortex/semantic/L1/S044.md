# Vericity SS.lv Scraping Pipeline

All in eu-north-1 (Stockholm), account 487042309472.

## Core scraping pipeline
1. **real-riga-ss-new-urls** (3GB, 5min) -- discover new listing URLs
2. **real-riga-ss-url-cache** (128MB, 15min) -- cache/deduplicate URLs
3. **real-riga-ss-data-scrape** (1GB, 15min) -- scrape listing data
4. **ss-transform-cleaner** (1GB, 5min) -- transform and clean data
5. **Azure-database** (1GB, 10min) -- upload to dbo.listings_ss

## Image pipeline
