# S063: Justs Statistical Writing Style

**Domain:** Statistics / Academic Writing
**Source:** rezultati_dora.pdf (Latvian medical thesis) + CGKont.JCeplis.Thesis.Final.pdf (SSE Riga bachelor thesis)

## Inline Stats Format
- Always parenthetical: (S = 12.5, n = 70), (U = 48.5, p = 0.2945), (Chi-Kv. = 5.69, p = 0.1277)
- Mean as M, SD as S. CI in square brackets: [59.6, 65.5] or CI(95%)
- p-vertiba not p-value in Latvian. Never naked numbers.

## Table Format
- Multi-group: Skaits, Vid., SD, Min, Max, CI (95%), Mediana, 1.kvartile, 3.kvartile
- Pairwise tests: Grupa 1, Grupa 2, U-statistika, p-vertiba (simple 4-col)
- Numbered: "X.Y. tabula" in Latvian, "Figure N" in English. Captions ABOVE tables.

## Interpretation Pattern
- Always interpret immediately after stats
- Template: "Rezultati liecina, ka [finding] (stat = X, p = Y)."
- Uses "statistiski nozimigs/a" (not "butisks")
- Acknowledges limitations inline: "autors atzist, ka nozimiguma trukums var rasties..."
- States what WOULD change with more data

## Latvian Terminology
Chi-Kvadratu tests, Mann-Whitney U tests, p-vertiba, videjais vecums, mediana, kvartile, standartnovirze (S), ticamibas intervals CI(95%), statistiski nozimigs, Balstoties uz [test] rezultatiem, Tiek secinats ka, Tiek pielietots [test] lai [purpose]. "Skat." for see/ref.

## English Academic Style
- Subscript notation: M_HML = 25.11%, S_HML = 17.7%
- Cohen d inline. Cronbach alpha with threshold justification
- OLS regressions numbered: regression (1), regression (2)
- Percentage point (pp) for coefficients. APA citation style.

## Sentence Structures
- Latvian: "Salidzinot [groups], tika pielietots [test]. Rezultati liecina (stat = X, p = Y), ka [interpretation]."
- English: "Results show that [finding] (p = X, F = Y). However, since [caveat], authors conclude that [adjusted conclusion]."
- Always hedges non-significant results with sample size caveat.

**Apply when:** Writing statistics sections for Beatrise thesis, any Latvian academic output, or English research papers.
