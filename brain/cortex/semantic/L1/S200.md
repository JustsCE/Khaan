Rule 11 (bypass-grant-block) is non-bypassable and blocks python3 -c commands containing bypass/state-write patterns (bypass_expiry, write_state, read_state). This prevents self-granting gate bypasses. The bootstrap lock is sealed — only Justs can grant bypasses via HMAC token from .bypass.secret.
