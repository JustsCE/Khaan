# og Brain — Beatrise Dataset — Non-standard Robson 2A1/2A2 Split

Critical data-structure fact specific to this dataset (Robsona Grupas - dati.xlsx, sheet DATA 2).

## Actual coding in this dataset
- Robsona 2A1 (n=176) = ALL successful inductions. Every row has Dzemdību sākums = "Indukcija" AND Ķeizargrieziens = "Nē, vag. dzemdības".
- Robsona 2A2 (n=119) = ALL failed inductions. Every row has Dzemdību sākums = "Indukcija" AND Ķeizargrieziens = "Jā, neatliekams".

This is NOT the ISO Robson classification (where 2A = induced and 2B = pre-labour CS within nullipara-term-cephalic). The study uses 2A1/2A2 as a success/failure split within induced nullipara.

## Implication for LR
- Failed-induction primary model = Robson 2A pooled (n=295) with outcome = 2A2 membership.
- No separate LR for 2A1 or 2A2 alone — each has zero outcome variance.
- Robson 4 inductions (229) all succeeded -> no emergency CS in R4 -> R4 LR outcome must be any CS (Operativus), not emergency CS.
- Robson 5 induced subset: 26/42 failed inductions.

## Counts — induced + emergency CS across whole dataset
- Overall: 199 failed inductions (90 Virssvars + 109 Adipozi) out of 607 induced (322 V + 285 A).
- Distribution: 119 -> Robson 2A2, 26 -> Robson 5, 54 -> Robson groups not in the study (6-10).
