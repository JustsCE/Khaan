# Norde Car Dealership Financial Reporting

Tētis (Reinis Ceplis) manages Norde car dealership group financial reporting using Excel + Power BI.

## Structure
Unit sheets: Serviss, Krāsotava, Apakšdīleri, Auto Noma, Auto pārdošana, Kopējie, Administratīvie, NORDE KOPĀ.
Each has MoM comparison, YTD, 13-month rolling blocks.

## v4 Rebuild (April 2026, by Tētis using Claude)
- Scanned 122,695 cells, found 3,507 cross-references to 6 legacy dump sheets
- Hardcoded all historical data, deleted dump sheets: 2.4 MB → 540 KB (77% smaller)
- Added automation: 00_INSTRUCTIONS, 01_CONTROL (month selector), 02_PBI_IMPORT, 03_PBI_MAPPING
- Serviss converted to fully dynamic formulas — change month in one cell, everything re-pivots
- Remaining 7 unit sheets still hardcoded, conversion is mechanical once Serviss pattern proven

## Key Architecture
- Prior-year data: hardcoded once year closes (no live links to dumps)
- 02_PBI_IMPORT: name-match lookup fixes Power BI "omits empty rows" issue
- 03_PBI_MAPPING: adding new P&L line = one-row edit instead of workbook-wide refactor
- Power BI should emit data-64 matrix format (all units × months × P&L lines in one export)

## Roadmap
