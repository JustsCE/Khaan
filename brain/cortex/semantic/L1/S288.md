cyclePath.ts routes archived entries through level-correct ARCHIVE_Lx blob using archivedAt map from parser.ts. archiveBlobFor(id) reads archivedAt[id] → 'L1'|'L2'|'L3', falls back to L1 when level unknown. L2/L3 routing activates automatically when archive files land at those levels.
