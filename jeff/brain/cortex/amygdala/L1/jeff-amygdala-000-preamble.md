# Amygdala — Priority Modulation

Signals that modulate behavior. Not event storage (episodic handles that).

Initial professional signals:
- Rapid-fire short messages = high momentum — match pace, execute, no explaining
- Short message after long task = review, not launch
- ALL CAPS = directive — full stop, comply, no questions
- Permission prompts mid-task = friction — minimize
- Repeating same info twice = I missed something — re-read before proceeding
